# Changelog — SinType Desktop

All notable changes to SinType Desktop will be documented here.

-----

## [v2.0] — 2026 · Current Release 🚀

### ✨ Added

- **Mobile Sync System** — Phone as wireless keyboard, touchpad & file transfer via WiFi
- **macOS-inspired Toolbar UI** — Complete visual overhaul with modern macOS look & feel
- **Local Server System** — FastAPI-based local server for mobile connectivity
- **Easy Mapping Editor** — Custom Singlish mapping edit & save without code
- **QR Code Pairing** — Instant mobile pair via QR scan
- **File Transfer** — WiFi-based file sharing between phone and PC
- **Add to Home Screen** — Mobile interface installable as PWA

### 🔧 Improved

- Toolbar completely redesigned — cleaner, more professional
- Mobile connector stability improvements
- Singlish engine performance optimization

-----

## [v1.1] — 2026 · Toolbar Upgrade

### ✨ Added

- **Minimalist Toolbar** — Brand new clean, modern toolbar design
- Improved mode switching UI (F9 / F10 / F11 visual indicators)
- Better system tray integration

### 🔧 Improved

- First version toolbar replaced with fully redesigned minimalist UI
- Cleaner visual hierarchy and layout
- Reduced visual clutter for a distraction-free experience

### 🐛 Fixed

- Minor UI rendering issues from v1.0
- Hotkey conflict edge cases

-----

## [v1.0] — 2026 · Initial Release 🎉

### ✨ Added

- **Core Singlish → Sinhala conversion engine**
- **Unicode Mode** `F9` — Modern Sinhala fonts (Noto Sans Sinhala, Abhaya Libre)
- **Legacy FM Font Mode** `F11` — FM Abhaya, FM Gemunu, FM Malithi support
- **English Mode** `F10` — Transliteration pause
- **Global Hotkey System** — Works in any Windows application
- **30-Day Free Activation** — Key generation via sintype.lk
- **100% Offline** — Works without internet after activation
- Windows 10 / 11 support
